# Model Controller API and Actions

The Model Controller Class methods are generated by the [Rhodes Application Generator](/rhodes/generator). These actions, coded in the model-name_controller.rb file for the generated model, perform basic CRUD (create, read, update and delete) on the Rhodes model object. For more information, refer to [Model/View/Controller](/rhodes/application#modelviewcontroller).

<table border="1">
<tr>
	<td><code>:create</code></td>
	<td>Creates a Rhodes model object.</td>
</tr>
<tr>
	<td><code>:delete</code></td>
	<td>Deletes a Rhodes model object.</td>
</tr>
<tr>
	<td><code>:edit</code></td>
	<td>Edits a Rhodes model object using the edit.erb View file.</td>
</tr>
<tr>
	<td><code>:index</code></td>
	<td>Lists all Rhodes model objects using the index.erb View file.</td>
</tr>
<tr>
	<td><code>:new</code></td>
	<td>Creates a new Rhodes model object using the new.erb View file.</td>
</tr>
<tr>
	<td><code>:show</code></td>
	<td>Views a Rhodes model object using the show.erb View file.</td>
</tr>
<tr>
	<td><code>:update</code></td>
	<td>Updates the attributes of a Rhodes model object.</td>
</tr>
</table>
